Este sistema solicita ao usuário o CEP e retorna Estado, Cidade e Rua que o usuário reside.

1 - Informar Nome
2 - Informar Idade
3 - Informar o CEP


Informações adicionais: 
Só poderá fazer a consulta usuários acima de 18 anos.

O CEP deverá conter 8 digitos.
